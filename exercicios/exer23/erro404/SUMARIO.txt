💡 EXEMPLOS PRÁTICOS INCLUSOS
═══════════════════════════════════════════════════════════════════════════════

EXEMPLO 1: Background simples com imagem
   background-image: url('imagens/sua-imagem.png');
   background-size: cover;
   background-position: center;

EXEMPLO 2: Repetir padrão horizontalmente
   background-image: url('imagens/linha.png');
   background-repeat: repeat-x;
   background-size: auto;

EXEMPLO 3: Efeito parallax
   background-image: url('imagens/fundo.jpg');
   background-attachment: fixed;
   background-size: cover;

EXEMPLO 4: Múltiplas camadas
   background-image: 
       url('imagens/foreground.png'),
       url('imagens/background.jpg');
   background-size: 100% 100%, cover;

EXEMPLO 5: Gradiente + Imagem
   background-image: 
       linear-gradient(90deg, rgba(0,0,0,0.3), rgba(0,0,0,0.3)),
       url('imagens/foto.jpg');
   background-size: 100% 100%, cover;

📞 SUPORTE RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: Imagem não aparece
   SOLUÇÃO:
   1. Verifique o caminho (imagens/seu-arquivo.jpg)
   2. Confirme que o arquivo existe
   3. Use DevTools (F12) para ver erros
   4. Defina background-color como fallback

PROBLEMA: CSS não muda
   SOLUÇÃO:
   1. Pressione Ctrl+Shift+Delete (limpar cache)
   2. Recarregue com Ctrl+F5
   3. Confirme que salvou (Ctrl+S)
   4. Feche e abra novamente no navegador

PROBLEMA: Layout quebrado no mobile
   SOLUÇÃO:
   1. Abra DevTools (F12)
   2. Clique em "Toggle device toolbar"
   3. Teste diferentes tamanhos
   4. Verifique media queries em style.css
