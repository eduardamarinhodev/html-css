## 💾 Salvando Mudanças

### No VS Code:

1. Edite `style.css`
2. Pressione `Ctrl+S` para salvar
3. Volte ao navegador
4. Pressione `F5` para recarregar

### Teste também com DevTools (F12):

1. Clique direito → Inspecionar
2. Modifique estilos ao vivo
3. Veja mudanças instantaneamente

---

## 🎨 Personalizando

### Mudar cores do gradiente:

Abra `style.css`, localize `background:` na seção `.background-principal` e mude os valores hex (#667eea, #764ba2)

### Adicionar sua própria imagem:

1. Salve a imagem em `imagens/`
2. No CSS, mude: `background-image: url('imagens/sua-imagem.jpg')`
3. Teste diferentes `background-size`

### Mudar a mensagem 404:

Abra `index.html` e edite o texto dentro das tags `<h1>`, `<p>`, etc

---

## 🐛 Se Algo Não Funcionar

1. **Imagem não aparece?**
   - Verifique o caminho do arquivo
   - Use DevTools (F12) para ver erros
   - Confirme que a imagem existe

2. **CSS não muda?**
   - Pressione `Ctrl+Shift+Delete` para limpar cache
   - Recarregue a página com `Ctrl+F5`
   - Confirme que salvou o arquivo

3. **Layout quebrado?**
   - Abra em navegador diferente
   - Verifique se há erros no console (F12)
   - Confirme tamanho da janela (teste mobile com F12)

## ✨ Dicas de Ouro
